# 🌐 智能日记本 - 浏览器版

本项目已改为本地浏览器界面，摒弃桌面 Tkinter 方案，使用 Flask 在浏览器中打开，界面风格接近微软官网的清爽、沉稳、模块化设计。

## ✨ 核心功能

- **浏览器打开**，无需桌面应用程序窗口
- **自动定位与天气**：获取当前城市并展示实时天气信息
- **微软官网样式**：清爽蓝白风格、简洁导航、卡片式布局
- **本地保存日记**：保存为 JSON 和 TXT 文件，并支持下载
- **选择城市**：支持内置中国主要城市列表，手动切换天气

## 🛠️ 新界面说明

- 主页通过浏览器访问
- 页面左侧显示天气与城市控制
- 右侧为日记编辑区，支持标题、正文输入和保存
- 保存后可直接下载生成文件

## 📦 安装步骤

### 1. 下载或克隆项目
```bash
git clone <repository-url>
cd 写日记软件
```

### 2. 安装依赖
```bash
pip install -r requirements.txt
```

### 3. 启动应用
```bash
python main.py
```
或
```bash
python start.py
```

浏览器将自动打开本地界面，地址为 `http://127.0.0.1:5000`。

## 🔧 运行提示
